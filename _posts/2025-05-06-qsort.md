# qsort函数
函数原型
`void qsort(int *base, size_t num, size_t size, int (*campar)(const void *, const void *))`
是标准库<stdio.h>中的函数，用于快速排序

使用示例
```
//qsort的参数函数
int cmp(const void *a, const void *b)
{
    return *(int *)a - *(int *)b;
}
int main()
{
    int *array = (int *)malloc(sizeof(int) * num);
    qsort(array, num, sizeof(int), cmp);
}
```
排序有时可以用于消除重复元素
